# ログ管理設計書

管理画面内での操作ログの管理ができる。
 
## ユースケース図
![ユースケース図：ログ管理](../../../svg/use_case/baser-core/dblogs.svg)

 
## 機能

### 検索
キーワードとユーザーでログの検索を行う事ができる。

### 表示件数設定
一覧への表示件数を変更できる。

### ページング
ページネーションによりページ移動ができる。
