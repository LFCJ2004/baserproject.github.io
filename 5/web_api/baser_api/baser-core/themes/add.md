# テーマの新規追加

新しいテーマをアップロードします。

## 実行に必要な権限

```
システム管理者以上
```

## リクエスト
```
POST /baser/api/baser-core/themes/add.json
```

### パスパラメーター

| パラメーター名 | 型     | 必須  | 内容         |
|---------|-------|-----|------------|
| file　   | 文字列	 | ●   | テーマの圧力ファイル |

## レスポンス例

### レスポンスボディ

```json
{
  "message": "テーマファイル「BcThemeSample2」を追加しました。",
  "theme": "BcThemeSample2"
}

```
