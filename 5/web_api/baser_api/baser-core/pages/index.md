# 固定ページ一覧の取得

固定ページ情報の一覧を取得します。

### 実行可能な権限
```
すべてのユーザー：公開された情報のみ
ログインユーザー以上：全て
```
 
### リクエスト
```
GET /baser/api/baser-core/pages.json
``` 

### クエリパラメーター

| パラメーター名           | 型 | 内容                                       |
|-------------------| --- |------------------------------------------|
| status            | 文字列 | publish：公開されたコンテンツフォルダーのみ<br>all：全て |
| limit             | 数値 | 取得件数                                     |
| page              | 数値 | ページ数                                     |
| contents   | 文字列 | 本文（あいまい検索）                      |
| draft     | 文字列 | 草稿（あいまい検索）                        |
