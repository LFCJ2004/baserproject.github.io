# メールの送信テスト実行

メールの送信テストを実行します。

### 実行可能な権限
```
システム管理者以上
```

### リクエスト
```
POST /baser/api/baser-core/site_configs/check_sendmail.json
``` 

### リクエストボディ

| パラメーター名       | 型     | 必須    | 内容      |
|---------------|-------|-------|---------|
| email         | 文字列 | ●     | メールアドレス |

### レスポンス例
#### レスポンスボディ
```json
{
  "message": ""
}

```
