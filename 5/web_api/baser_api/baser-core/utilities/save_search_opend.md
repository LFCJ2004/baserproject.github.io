# 検索ボックスの表示状態を保存する

検索ボックスの表示状態を保存します。

### 実行可能な権限
```
システム管理者（システム管理者の追加はユーパーユーザーのみ）
```

### リクエスト
```
POST /baser/api/baser-core/utilities/save_search_opened/{key}/{open}.json
```

### パスパラメーター

| パラメーター名   | 型   | 必須  | 内容    |
|-----------|-----|-----|-------|
| key        | 文字列  | ●   | キー    |
| open        | 数字  | ●   | 1 Or '' |

### レスポンス例
#### レスポンスボディ
```json
{
  "result": true
}

```
