# コンテンツフォルダーのリストの取得

指定したサイトのフォルダリストを取得

### 実行可能な権限
```
ログインユーザー以上
```
 
### リクエスト
```
GET /baser/api/admin/baser-core/contents/get_content_folder_list/{siteId}.json
``` 

### パスパラメーター

| パラメーター名         | 型   | 内容     |
|-----------------|-----|--------|
| siteId | 数値  | サイトのID |

### レスポンス例
#### レスポンスボディ
```json
{
    "list": {
        "3": "会社案内",
        "4": "サービス"
    }
}
```
