# ユーザーグループの再構築

ユーザーグループを指定してアクセスルールを再構築します。

### 実行可能な権限
```
システム管理者以上
```

### リクエスト
```
POST baser/api/admin/baser-core/permission_groups/rebuild_by_user_group/{userGroupId}.json
``` 

### パスパラメーター

| パラメーター名   | 型   | 必須  | 内容          |
|-----------|-----|-----|-------------|
| userGroupId        | 数値  | ●   | ユーザーグループのID |

### レスポンス例
#### レスポンスボディ
```json
{
  "message": "アクセスルールの再構築に成功しました。"
}
```
