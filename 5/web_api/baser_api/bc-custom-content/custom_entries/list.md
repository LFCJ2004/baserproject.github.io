# カスタムエントリーのリスト取得

カスタムエントリーのリストを取得します。

## 実行に必要な権限

```
ログインユーザー以上
```

## リクエスト
```
GET baser/api/admin/bc-custom-content/custom_entries/list.json
```

### クエリパラメーター

| パラメーター名   | 型   | 必須  | 内容                     |
|-----------|-----|-----|------------------------|
| custom_table_id            | 数値 |  ● | カスタムテーブルID |

## レスポンス例

### レスポンスボディ

```json
{
  "entries": {
    "3": "工業",
    ...
  }
}
```
