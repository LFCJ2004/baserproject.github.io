# ブログタグの削除

指定したブログタグを削除します。

### 実行可能な権限
```
システム管理者以上
```

### リクエスト
```
DELETE /baser/api/admin/bc-blog/blog_tags/{blogTagId}.json
``` 

### パスパラメーター

| パラメーター名   | 型   | 必須  | 内容                |
|-----------|-----|-----|-------------------|
| blogTagId        | 数値  | ●   | ブログタグID              |

### レスポンス例
#### レスポンスボディ
```json
{
  "blogTag": {
    "id": 4,
    "name": "vietnam",
    "created": "2023-03-30T17:30:29+09:00",
    "modified": "2023-03-30T17:30:58+09:00"
  },
  "message": "ブログタグ「vietnam」を削除しました。"
}

```
