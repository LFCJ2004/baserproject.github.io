# ブログ記事の一括処理

指定したブログ記事について一括にて次の処理ができます。

- 削除
- 公開状態に設定
- 非公開状態に設定。

### 実行可能な権限
```
システム管理者以上
```

### リクエスト
```
POST /baser/api/bc-blog/blog_posts/batch.json
``` 

### リクエストボディ

| パラメーター名       | 型     | 必須    | 内容                                                 |
|---------------|-------|-------|----------------------------------------------------|
| batch_targets | 配列   | ●     | ブログ記事ID                                            |
| batch         | 文字列 | ●     | 公開する: publish<br>非公開にする: unpublish<br>削除する: delete |

### レスポンス例
#### レスポンスボディ
```json
{
  "message": "一括処理が完了しました。"
}

```
