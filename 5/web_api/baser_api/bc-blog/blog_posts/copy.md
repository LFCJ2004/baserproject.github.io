# ブログ記事のコピー

指定したブログ記事をコピーします。

### 実行可能な権限
```
システム管理者以上
```

### リクエスト
```
POST /baser/api/bc-blog/blog_posts/copy/{blogPostId}.json
``` 

### リクエストボディ

| パラメーター名   | 型   | 必須  | 内容                |
|-----------|-----|-----|-------------------|
| blogPostId        | 数値  | ●   | ブログ記事ID              |
