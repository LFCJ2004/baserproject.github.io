# ブログ記事の非公開

指定したブログ記事を非公開にします

### 実行可能な権限
```
システム管理者以上
```

### リクエスト
```
POST /baser/api/bc-blog/blog_posts/unpublish/{blogPostId}.json
``` 

### パスパラメーター

| パラメーター名   | 型   | 必須  | 内容                |
|-----------|-----|-----|-------------------|
| blogPostId        | 数値  | ●   | ブログ記事ID              |

### レスポンス例
#### レスポンスボディ
```json
{
  "message": "ブログ記事「ブログ機能について説明します」を非公開状態にしました。"
}

```
