# お気に入りの新規追加

新しいお気に入りを作成します。


## 実行に必要な権限

```
システム管理者以上
```

## リクエスト
```
POST /baser/api/bc-favorite/favorites.json
```

### リクエストボディ

| パラメーター名   | 型   | 必須  | 内容     |
|-----------|-----|-----|--------|
| user_id   | 数値  | ●   | タイトル |
| name   | 文字列 | ●　   | お気に入り名 |
| url   | 文字列 | 　   | リンク    |
| sort   | 数値 | 　   | 並び替え   |

## レスポンス例

### レスポンスボディ

```json
{
  "message": "お気に入り「お問い合わせ」を追加しました。",
  "favorite": {
    "sort": 3,
    "user_id": 1,
    "name": "お問い合わせ",
    "url": "/contact",
    "created": "2023-03-31T17:04:51+09:00",
    "modified": "2023-03-31T17:04:51+09:00",
    "id": 3
  },
  "errors": null
}

```
