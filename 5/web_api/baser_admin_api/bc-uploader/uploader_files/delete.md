# アップロードファイルの削除

既存のアップロードファイルを削除します

## 実行に必要な権限

```
システム管理者以上
```

## リクエスト
```
DELETE /baser/api/admin/bc-uploader/uploader_files/{uploaderFileId}.json
```

### パスパラメーター

| パラメーター名   | 型   | 必須  | 内容        |
|-----------|-----|-----|-----------|
| uploaderFileId   | 数値  | ●   | アップロードファイルID       |

## レスポンス例

### レスポンスボディ

```json
{
  "message": "アップロードファイル「img4.jpg」を削除しました。"
}
```
