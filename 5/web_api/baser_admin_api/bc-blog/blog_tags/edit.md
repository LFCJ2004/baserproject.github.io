# ブログタグの編集

新しいブログタグを編集します。


## 実行に必要な権限

```
システム管理者以上
```

## リクエスト
```
PATCH /baser/api/admin/bc-blog/blog_tags.json
```

### リクエストボディ

| パラメーター名   | 型   | 必須  | 内容                |
|-----------|-----|-----|-------------------|
| name   | 文字列 |     | ブログタグ名  |

## レスポンス例

### レスポンスボディ

```json
{
  "blogTag": {
    "id": 4,
    "name": "vietnam",
    "created": "2023-03-30T17:30:29+09:00",
    "modified": "2023-03-30T17:30:58+09:00"
  },
  "message": "ブログタグ「vietnam」を更新しました。",
  "errors": null
}

```
