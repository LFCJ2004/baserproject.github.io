# テーマフォルダの現在のテーマにコピー

対象のフォルダを現在のテーマの同階層にコピーします

## 実行に必要な権限

```
システム管理者以上
```

## リクエスト
```
POST /baser/api/admin/bc-theme-file/theme_folders/copy_to_theme.json
```

### リクエストボディ

| パラメーター名   | 型   | 必須  | 内容    |
|-----------|-----|-----|-------|
| theme   | 数値  | ●   | テーマ   |
| type   | 文字列 | ●　  | タイプ   |
| path   | 文字列 | ●　  | パス    |
| plugin   | 数値 | ●   | プラグイン |
| assets   | 数値 | ●   | アセットス |

## レスポンス例

### レスポンスボディ

```json
{
  "message": "コアフォルダ base_name_1.php を テーマ BcFront の次のパスとしてコピーしました。\\n。"
}
```
