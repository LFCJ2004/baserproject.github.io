# お気に入りの編集

指定したお気に入りを編集します。


## 実行に必要な権限

```
システム管理者以上
```

## リクエスト
```
PATCH /baser/api/admin/bc-favorite/favorites/{favoriteId}.json
```

### パスパラメーター

| パラメーター名   | 型   | 必須  | 内容                |
|-----------|-----|-----|-------------------|
| favoriteId        | 数値  | ●   | お気に入りID              |

### リクエストボディ

| パラメーター名   | 型   | 必須  | 内容     |
|-----------|-----|-----|--------|
| name   | 文字列 | ●　  | お気に入り名 |
| user_id   | 数値  |     | タイトル |
| url   | 文字列 | 　   | リンク    |
| sort   | 数値 | 　   | 並び替え   |

## レスポンス例

### レスポンスボディ

```json
{
  "favorite": {
    "id": 4,
    "user_id": 1,
    "name": "問い合わせ",
    "url": "/contact",
    "sort": 4,
    "created": "2023-03-31T17:16:57+09:00",
    "modified": "2023-03-31T17:21:15+09:00"
  },
  "message": "お気に入り「問い合わせ」を更新しました。",
  "errors": null
}

```
