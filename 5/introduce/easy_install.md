# 簡単インストール

簡単インストールとは、レンタルサーバー会社が用意した手順を踏めば、baserCMSを自身でアップロードすることなく、簡単にbaserCMSを利用できるようになる機能の一般的な呼称です。データベースの設定(ユーザ名・パスワードなど)がプリセットされていることも多いです。通常、レンタルサーバー会社のマニュアルに具体的な手順が記載してあります。

## 利用上の注意点
簡単インストールでインストールされるbaserCMSは、最新バージョンではない可能性があります。インストールされるバージョンは、各レンタルサーバー会社のHPをご覧ください。

## 簡単インストールが利用できるレンタルサーバー

2023年3月27日現在で簡単インストールが利用できるのは、以下のサーバーです。

| レンタルサーバー名          | サービス名  |
|-------------------------------|------------------------------|
| [ロリポップ！](https://lolipop.jp/) | [簡単インストール](https://lolipop.jp/manual/user/applications-bc/) |
| [ヘテムル](https://heteml.jp/)    | [簡単インストール](https://support.heteml.jp/hc/ja/articles/360042620133-baserCMS-%E3%81%AE%E3%82%A4%E3%83%B3%E3%82%B9%E3%83%88%E3%83%BC%E3%83%AB)                                                |
